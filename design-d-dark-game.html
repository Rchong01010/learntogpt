Learn by Doing

Every lesson is a hands-on challenge. Write real prompts. Get real feedback.

lesson-07-xml-tags.py +25 XP
Challenge: Rewrite this prompt using XML tags to separate instructions from data.
# Your current prompt (messy):
prompt = "Summarize this article and extract
key facts. The article is about AI safety..."

# Rewrite with XML structure:
prompt = """
<instructions>
Summarize and extract key facts.
</instructions>
<article>
</article>
"""
Hint: XML tags help Claude distinguish between your instructions and the data it should process. This pattern reduces hallucination and improves accuracy.
